 # 🚀 GitHub Profile Finder

A modern and responsive **GitHub Profile Finder** built using **HTML, CSS, and JavaScript**. This application uses the **GitHub REST API** to fetch and display public GitHub profile information in a clean and user-friendly interface.

## ✨ Features

* 🔍 Search any GitHub username
* 👤 Display profile picture and name
* 📝 View bio and profile details
* 👥 Followers & Following count
* 📂 Public repositories count
* 📍 Location and Company
* 🔗 Personal website/blog link
* 📅 GitHub account creation date
* ⭐ Display top repositories
* ⏳ Loading indicator
* ❌ Error handling for invalid usernames
* 📱 Fully responsive design

## 🔍 How It Works

1. Enter a GitHub username.
2. Click the **Search** button or press **Enter**.
3. The application fetches profile data from the GitHub API.
4. User information and repositories are displayed instantly.

---

## 🌐 API Used

GitHub REST API

```
https://api.github.com/users/{username}
```
